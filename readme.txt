== Description ==
Vintage minimalist theme. Includes social media icons (configurable from Appearance -> Customize).

* Responsive layout
* Retina-ready design, .svg illustrations and icons
* Social Media profiles
* Custom logo option
* Custom background option
* Yoast SEO breadcrumbs integration
* Optimized for performance

== Frequently Asked Questions ==

= How do I add the social media icons and links in the header? =

Visual Coffee allows you display links to your social media profiles.
To display the icons in header follow these steps:

1. Open Live Customizer by going to Appearance > Customize
2. Add your links in Social Media Icons section.
